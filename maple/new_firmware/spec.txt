===HEADERS AND CODES===
Header packets:
INIT      'I'
GET       'G'
SET       'S'

RESPONSE  'R'

END       0xFF

Device Codes:
AnalogInput 'A'
Cytron      'C'
Ultrasonic  'U'

===COMPUTER TO MAPLE COMMS===

INIT packet structure:
I
[DEVICE_COUNT]
[DEVICE_CODE] [DEVICE_BYTES]
[DEVICE_CODE] [DEVICE_BYTES]
...
[DEVICE_CODE] [DEVICE_BYTES]
0xFF

GET packet structure
G
//currently does not expect END, but safely accepts one

SET
S
[GLOBAL_DEVICE_INDEX] [BYTES_FOR_DEVICE]
[GLOBAL_DEVICE_INDEX] [BYTES_FOR_DEVICE]
...
[GLOBAL_DEVICE_INDEX] [BYTES_FOR_DEVICE]
0xFF

===MAPLE TO COMPUTER COMMS===

RESPONSE packet structure
R
[DEVICE OUTPUT BYTES]
[DEVICE OUTPUT BYTES]
...
[DEVICE OUTPUT BYTES]
0xFF

===DEVICE SPECIFICATIONS===

=ANALOG_INPUT=
initialization bytes:
byte    [pin]

response bytes:
uint16  [analog_input]

=CYTRON=
initialization bytes:
byte    [dirPin]
byte    [pwmPin]

set bytes:
int16   [speed]

=ULTRASONIC=
initialization bytes:
byte    [triggerPin]
byte    [echoPin]

response bytes
uint16  [delay_micros]
